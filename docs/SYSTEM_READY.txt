WORKFLOW 3: Python Analysis
───────────────────────────
```python
from categorisation import load_processed_data
from airfoil_generation import generate_and_validate_airfoil

# Load data
data = load_processed_data()
df = data['categorized']

# Get diving birds
divers = df[df['flight_category'] == 'diving']

# Generate airfoil for top performer
best = divers.nlargest(1, 'efficiency_index').iloc[0]
upper, lower, meta, valid = generate_and_validate_airfoil(best)

print(f"Best diver: {best['species']}")
print(f"Efficiency: {best['efficiency_index']:.3f}")
print(f"Thickness: {meta['max_thickness']:.3f}")
```


💾 FILE FORMATS
════════════════════════════════════════════════════════════════

.dat files (airfoils/)
─────────────────────
• Standard CFD format (XFOIL, ANSYS)
• 200 points per airfoil
• Closed loop (TE to TE, clockwise)
• Header with complete metadata
• Normalized coordinates (x/c, y/c)

.csv files (control_points/)
────────────────────────────
• Bézier control points
• 12 points per surface
• Parametric regeneration
• CSV format for easy editing
• Complete metadata in comments

🆘 TROUBLESHOOTING
════════════════════════════════════════════════════════════════

Q: Dashboard won't load?
   → Check ports 8501/8502 aren't in use
   → Run: lsof -ti:8501 | xargs kill -9

Q: Can't find airfoil files?
   → Check: OUTPUT/airfoils/ directory
   → Run generation: python airfoil_generation.py

Q: Want to generate more airfoils?
   → Run: python airfoil_generation.py
   → It will continue from where it stopped

Q: How to export for CFD?
   → .dat files in OUTPUT/airfoils/ are ready to use
   → Import directly into XFOIL/ANSYS
